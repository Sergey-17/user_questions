Создай фронтенд-приложение для проекта. 

Условия:
- Фронтенд хранится в отдельной папке frontend, независимо от бэкенда;

- Создай отдельный Dockerfile для фронтенда;

- В docker-compose.yml добавь сервис frontend, который билдит и запускает фронтенд (команды: npm install и npm run dev);

- Максимальная ширина контента — 800px с центрированием по экрану;

- На странице должен быть: заголовок крупным современным шрифтом, поле ввода ссылки, кнопка "Отправить";

- По нажатию кнопки выполняется POST-запрос на эндпоинт бэкенда /analyze-site;

- При получении ответа данные должны красиво анимироваться и появляться в виде блоков;

- Сделай глобальный лоадер (например, спиннер) с подписью "Анализируем сайт...";

- Обработай ошибки;

-  Используй Axios для запросов;

- Добавь лёгкую адаптивность.
